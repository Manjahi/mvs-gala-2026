# 🏊 MVS Gala 2026 - Quick Start Guide

## 🚀 Quick Setup (5 Minutes)

### 1. Admin URL (Your Laptop)

```
https://YOUR_USERNAME.github.io/mvs-gala-2026/Swim.html
```

- Press **'P'** to hide admin button
- Press **Ctrl+Shift+M** to show admin if hidden

### 2. Public URL (TV/Projector)

```
https://YOUR_USERNAME.github.io/mvs-gala-2026/Swim.html?view=public&t=TOKEN
```

- Open Admin Panel → Generate QR Code
- Scan with TV/Tablet
- Viewers see live updates only

---

## ⏱️ Entering Scores

1. Click **"Event Status"** tab
2. Click on an event name
3. Enter times for each house:
   - `MM:SS.MS` (Minutes : Seconds . Milliseconds)
4. Check **DNS/DNF** if swimmer didn't finish
5. Click **"Save Scores"**

✅ TV updates automatically!

## 🎥 Adding Videos

1. Open Admin Panel
2. Go to **"Manage Videos"**
3. Paste YouTube URL
4. Click **"Add Video"**

Videos autoplay during rotation.

---

## 🆘 Emergency Backup

If Firebase fails:

1. Export data: Admin Panel → **"Export All Data"**
2. Save JSON file
3. Can import later to restore
